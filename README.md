# OpenSoulInc — лендинг

Интерактивный одностраничный сайт компании **OpenSoulInc** (организация лечебно-диетического
и общественного питания) на React + Vite + Framer Motion.

## Возможности

- 🌍 Первый экран: вращающаяся Земля (canvas, спутниковый снимок NASA Blue Marble) → плавный
  переход в карту Казахстана с реальными объектами компании
- 🍽️ «Провал» в Астану: накрытый стол, который сервируется по мере скролла (реальное фото
  проявляется по одному блюду)
- 🔄 «Путь тарелки» — 5 процессов (стол → кухня → очистка → логистика → заявка), каждый
  собирается анимационно по скроллу
- 🌗 Тёмная / светлая тема (по умолчанию тёмная), выбор запоминается
- 🌐 Три языка: RU (по умолчанию), KZ, EN — переключение на лету
- ✨ Кастомный курсор, магнитные кнопки, 3D-tilt карточки, счётчики, таймлайн, отзывы
- 📱 Адаптив под все экраны, оптимизация производительности (анимации вне экрана не крутятся)

## Запуск

```bash
npm install
npm run dev      # http://localhost:5173
npm run build    # прод-сборка в dist/
npm run preview  # предпросмотр сборки
```

## Структура

- `src/components/` — секции и эффекты (Cosmos, Astana, Journey, Sections, fx, scenes)
- `src/i18n.jsx` — все переводы (RU/KZ/EN)
- `src/theme.jsx` — переключение темы
- `src/data/` — контур и данные карты Казахстана (Natural Earth)
- `public/` — изображения (логотип, спутниковые снимки, фото блюд)

## Замена контента

- Тексты — `src/i18n.jsx`
- Логотип — `public/logo.png`
- Фото блюд для «Пути тарелки» — `public/scenes/`, накрытый стол — `public/food.jpg`
- Объекты на карте — `src/data/kz.js`
